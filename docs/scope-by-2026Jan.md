# 2026年1月末までの開発スコープ

## 目的

2026年3月末にチームみらいが政治資金報告書を提出する際に、本アプリケーションを使用できるようにする。
そのために、チームみらいが実際に必要とする項目にフォーカスして機能開発を行う。

## 前提条件

- 2025年のチームみらいは政治資金パーティーを開催していないため、パーティー関連の様式は不要
- 報告対象となる資産（不動産・金銭信託等）を保有していないため、資産関連の様式は不要
- 寄附のあっせん、政党匿名寄附も実績がないため対応不要
- 報告書は政治資金規正法に基づく適切なXMLフォーマットで出力する必要がある

## 対象様式

### 対応必須の様式

| 様式タグ | 様式名 | 説明 |
|---------|-------|------|
| SYUUSHI_UMU_FLG | 様式有無フラグ | 各様式のデータ有無を示す51桁の文字列 |
| SYUUSHI07_01 | 第14号様式（その1） | 団体基本情報（団体名、代表者、会計責任者、事務所所在地等） |
| SYUUSHI07_02 | 第14号様式（その2） | 収支総括表（収入総額、支出総額、繰越額、寄附内訳等） |
| SYUUSHI07_03 | 第14号様式（その3） | 事業による収入（機関紙発行、売店販売等） |
| SYUUSHI07_04 | 第14号様式（その4） | 借入金の明細 |
| SYUUSHI07_05 | 第14号様式（その5） | 本部または支部から供与された交付金 |
| SYUUSHI07_06 | 第14号様式（その6） | その他の収入（10万円以上/未満の区分あり） |
| SYUUSHI07_07 | 第14号様式（その7） | 寄附の明細（個人・法人・政治団体の3区分） |
| SYUUSHI07_14 | 第14号様式（その14） | 経常経費の支出（光熱水費・備品消耗品費・事務所費） |
| SYUUSHI07_15 | 第14号様式（その15） | 政治活動費の支出（組織活動費・選挙関係費・宣伝事業費等9種類） |
| SYUUSHI07_16 | 第14号様式（その16） | 本部または支部に対する交付金の支出 |

### 対応不要の様式

以下の様式は、2025年のチームみらいでは該当する取引・資産が存在しないため対応しない。

| 様式タグ | 様式名 | 対応不要の理由 |
|---------|-------|--------------|
| SYUUSHI07_08 | 第14号様式（その8） | 寄附のあっせんによるもの → 実績なし |
| SYUUSHI07_09 | 第14号様式（その9） | 政党匿名寄附 → 実績なし |
| SYUUSHI07_10 | 第14号様式（その10） | 政治資金パーティーの対価に係る収入 → パーティー未開催 |
| SYUUSHI07_11 | 第14号様式（その11） | パーティー対価の支払者明細 → パーティー未開催 |
| SYUUSHI07_12 | 第14号様式（その12） | パーティー対価のあっせん → パーティー未開催 |
| SYUUSHI07_17 | 第14号様式（その17） | 資産等の項目別内訳の有無 → 報告対象資産なし |
| SYUUSHI07_18 | 第14号様式（その18） | 資産等の項目別内訳の明細 → 報告対象資産なし |
| SYUUSHI07_19 | 第14号様式（その19） | 不動産に関する使用の状況 → 不動産なし |
| SYUUSHI07_20 | 第14号様式（その20） | 宣誓書 → 手動対応可能 |
| SYUUSHI08 | 第15号様式 | 訂正等届出書 → 初回提出時は不要 |
| SYUUSHI08_02 | 第16号様式 | 解散届出書 → 解散予定なし |
| SYUUSHI_KIFUKOUJYO | 寄附金控除団体情報 | 寄附金控除対象団体ではない |
