# 日次寄附グラフ関連コードの削除（Phase 3）

## 目的

**開発者が**、使用されていないコードを保守するコストを削減するため。

## 調査結果

`GetDailyDonationUsecase` および関連コードは**現在どこからも参照されていない**。

- loader からの参照なし
- app（ページ・コンポーネント）からの参照なし
- テストファイルと定義ファイルのみ存在

## 方針

DDDリファクタリングではなく、**未使用コードを削除する**。

必要になった時点で、DDD構造に従って新規実装する。
