# SYUUSHI07_13・SYUUSHI07_16 実装設計

## 1. 概要

本設計書は以下の2つのシートの実装を対象とする：

- **SYUUSHI07_13**（第14号様式 その13）: 支出項目別金額の内訳（総括表）
- **SYUUSHI07_16**（第14号様式 その16）: 本部または支部に対する交付金の支出

### 1.1 SYUUSHI07_13の背景

SYUUSHI07_13は、経常経費（シート14）と政治活動費（シート15）の合計値を集約し、Excelテンプレートの「シート21（その13）」に対応する。

- 公式仕様書には記載がないため、エクスポートソフトから出力されたXMLサンプル（`data/reports/minimum_valid.xml`）を元に推測した仕様である
- シート14/15の内訳合計と整合性が求められ、不整合があるとExcelテンプレートでバリデーションエラー（E185等）が発生する

### 1.2 SYUUSHI07_16の背景

SYUUSHI07_16は、本部または支部に対する交付金の支出の明細を格納する。scope-by-2026Jan.mdで対応必須の様式として指定されている。

### 1.3 関連シート

| シート | 内容 | 関係 |
|--------|------|------|
| SYUUSHI07_13 | 支出項目別金額の内訳（本設計） | 集約元 |
| SYUUSHI07_14 | 経常経費の支出（光熱水費・備品消耗品費・事務所費） | 合計値の元データ |
| SYUUSHI07_15 | 政治活動費の支出（9種類） | 合計値の元データ |
| SYUUSHI07_16 | 本部または支部に対する交付金の支出（本設計） | 明細データ |

## 7. SYUUSHI_UMU_FLG対応

### 7.1 フラグ位置

51桁フラグのうち、本設計に関連するもの:

| 桁位置 | シート | 判定条件 |
|--------|--------|----------|
| 21 | その13（支出項目別内訳） | 支出総額が0でない場合（人件費のみの場合も含む） |
| 34 | その16（本支部交付金） | 本支部交付金データがある場合 |

### 7.2 フラグ判定ロジック

```
// シート13: 支出が「０」の場合は不要、それ以外は出力
// 人件費のみの場合も考慮する必要がある（シート14にKUBUNがないため個別判定が必要）
shouldOutputSheet13 = (
  expenses.personnelExpenses.totalAmount > 0 ||
  ExpenseData.shouldOutputRegularExpenseSheet(expenses) ||
  ExpenseData.shouldOutputPoliticalActivitySheet(expenses)
)

// シート16
shouldOutputSheet16 = branchGrantExpenses.rows.length > 0 || branchGrantExpenses.totalAmount > 0
```

**注意**: 人件費はシート14にKUBUNがないため、`shouldOutputRegularExpenseSheet`では検出されない。
そのため、人件費のみがある場合でもシート13を出力するために、人件費の個別判定を追加する。

## 8. 人件費・交付金に関する注意

### 8.1 人件費（JINKENHI）

人件費は政治資金規正法上、特別な扱いを受けるカテゴリである。

**法的背景:**
- 10万円以上の支出であっても、支払先の氏名・住所の開示が不要
- プライバシー保護の観点から明細記載が免除されている

**XML仕様上の扱い:**
- シート14（SYUUSHI07_14）には人件費のKUBUNが存在しない
  - KUBUN1: 光熱水費
  - KUBUN2: 備品・消耗品費
  - KUBUN3: 事務所費
  - 人件費のKUBUNはない
- シート13（SYUUSHI07_13）の総括表にのみ合計額を記載

**データソース:**
- 会計データのPLカテゴリ「人件費」（`shared/accounting/account-category.ts`で定義）から取得
- `ExpenseData`に人件費専用のセクションを追加して集計

### 8.2 交付金（KOUFU列）

シート13のKOUFU列（例: JINKENHI_KOUFU）は「各支出項目のうち交付金から支出した額」を示す財源区分である。

- 現在の会計データには財源区分の情報がない
- 現時点ではnull（空タグ）として出力
- 将来的に財源トラッキング機能が追加された場合に対応

### 8.3 事業費内訳の合計

- SONOTA_JIGYO_GK（機関紙誌の発行その他の事業費）は以下4項目の合計:
  - HAKKOU_JIGYO_GK（機関紙誌の発行事業費）
  - SENDEN_GK（宣伝事業費）
  - KAISAI_GK（パーティー開催事業費）
  - SONOTA_GK（その他の事業費）
