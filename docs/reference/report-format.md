# 政治資金収支報告書 XMLデータフォーマット仕様書

> **注意**: このドキュメントは参考資料として作成されたものであり、内容の正確性を必ずしも保証するものではありません。実際の実装や提出の際は、必ず公式の仕様書や関連法規を確認してください。

## 概要

このファイルは、政治資金規正法に基づく収支報告書を電子申請するためのXMLデータフォーマット仕様を定義したものである。政治団体が総務省または都道府県選挙管理委員会に提出する収支報告書データの構造、各項目の定義、および入力規則を記載している。

| 項目 | 値 |
|------|-----|
| 帳票・ファイル名 | 収支報告書データ |
| ファイル種別 | XML |
| エンコーディング | Shift_JIS |

## XMLデータ構造

収支報告書XMLデータは、最上位要素 `<BOOK>` の中に各様式のデータがネストされる構造になっている。

### 主要な構成要素

| 要素 | 説明 |
|------|------|
| HEAD | ファイルのバージョン情報やアプリケーション情報を格納 |
| SYUUSHI_UMU_FLG | 51桁の文字列で、各様式のデータ有無を「0」（無）または「1」（有）で表現（詳細は下記参照） |
| SYUUSHI07_xx | 第14号様式（収支報告書の本体データ）。各様式は `<SHEET>` 要素で囲まれる |
| SYUUSHI08 | 第15号様式（訂正等の届出） |
| SYUUSHI08_02 | 第16号様式（解散届出） |
| SYUUSHI_KIFUKOUJYO | 寄附を受けた団体の情報（寄附金控除関連） |

### 明細行の構造

## 様式一覧

「様式」とは、政治資金規正法施行規則で定められた報告書の書式のことである。収支報告書は複数の様式で構成されており、団体の基本情報、収入・支出の内訳、資産状況など、それぞれ異なる内容を報告する。

- **第14号様式**: 収支報告書の本体（その1〜その20まで20種類）
- **第15号様式**: 訂正等届出書

## 各様式の項目定義

### HEAD（ヘッダ）

XMLファイルのメタ情報を格納する。ファイルのバージョンや作成元アプリケーションの情報が含まれる。

| 項番 | 項目名 | タグ名 | 設定条件 | 項目属性 | 項目長（最大） | フォーマット |
|------|--------|--------|----------|----------|----------------|--------------|
| 1 | 様式名 | HEAD | 必須 | - | - | - |
| 2 | ファイルバージョン | VERSION | 必須 | NUMBER | 8 | NULL不可 |
| 3 | ファイル作成アプリケーション名 | APP | 必須 | VARCHAR | 100 | NULL不可（「収支報告書」で開始される文字列） |
| 4 | ファイルフォーマット番号 | FILE_FORMAT_NO | 必須 | CHAR | 10 | "1"を設定する（将来ファイルレイアウトを変更し、過去のシステムで読込不可にする場合に変更） |
| 5 | 告示作成アプリ | KOKUJI_APP_FLG | 任意 | CHAR | 1 | 1：告示用の作成ソフトから出力　0：それ以外 |
| 6 | 会計帳簿作成ソフトバージョン | CHOUBO_APP_VER | 任意 | NUMBER | 8 | 作成元会計帳簿作成ソフトのバージョン |
