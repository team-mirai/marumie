# 3 収入・支出項目の分類基準表

## （収入）

| 項目 | 内容 |
|---|---|
| 1 党費又は会費 | 個人が負担する党費又は会費の合計金額及び納入した者の延人員（当該団体の構成員の合計金額）。なお、「法人その他の団体」からの党費は寄附となるので除く（法人等からの党費は寄附として記載される。）。 |
| 2 寄附 | |
| （1）個人又は会社 | 個人からその他の団体からの寄附（党費・会費として受けた金額を含む。） |
| （2）法人その他の団体からの寄附 | 法人その他の団体から受けた寄附（党費・会費として受けた金額を含む。）の寄附 |
| （3）政治資金団体からの寄附 | 政治団体として届出がある団体からの寄附 |
| （4）政党匿名寄附 | 政党及び政治資金団体が、街頭又は一般に公開される講演会若しくは集会において受けた1,000円以下の寄附。この寄附は、すべて禁止される政治団体が受けることはできないので5万円超（5万1円以上）の審判について記載する。 |
| 3 機関紙誌の発行その他の事業による収入 | 機関紙や機関雑誌の発行事業収入、政治資金パーティーの開催事業収入、新年会・忘年会等その他催し物の会費による収入をいう。具体的には、「○○機関紙発行事業」・「△△政治資金パーティー開催事業」・「□□立食会等名称を事業ごと、開催回ごとに記載する。なお、これら機関紙誌の発行その他の事業についでは、支出の2「政治活動費」・「（3）機関紙誌の発行その他の事業費」のいずれかの支出として掲載される。なお、これらの事業で「お祝い」等の会費以外の収入は、寄附となる。 |
| 4 借入金 | 個人又は金融機関等からの借入金 |
| 5 本部又は支部から供与された交付金に係る収入 | 本部・支部間又は支部間における（選挙管理委員会又は総務大臣へ届出がある交付金。）交付金・還付金・繰付金・寄附等により受けた「金銭以外のもの」による交付金の額 |
| 6 その他の収入 | 上記1から5に分類できない収入類で、例えば、預金利子や労務等の無償提供に係る寄附（他団体との交出に対応する「金銭以外のもの」によるこの項目相当分）の額をいう。1件10万円以上については明細記載する。 |

## （支出）

| 項目 | 内容 |
|---|---|
| 1 経常経費 | 総額を記載し、領収書は不要 |
| （1）人件費 | 政治団体の職員（機関紙誌の発行その他の事業に従事する者を除く。）に支払われる給料、報酬、扶養手当・通勤手当・住居手当その他の諸手当の額及び健康保険料その他の各種保険料の額をいう。 |
| （2）光熱水費 | 電気、ガス、水道の使用料及びこれらの計器使用料等をいう。 |
| （3）備品・消耗品費 | 机、椅子、ロッカー、複写機、自動車（事務所用に限る。）等の備品の購及び事務用品紙、封筒、ボールペン、勘定等の事務用品類、事務服、新聞、雑誌、ガソリン（事務所用自動車）等の消耗品の購入費をいう。 |
| （4）事務所費 | 事務所の賃借料（地代・家賃など）、公租公課、火災保険料、電話使用料、切手購入費等の各種保険料等、事務所の維持に通常必要とされるもの。<br>①資金管理団体は、内訳が必要（5万円超）<br>②国会議員関係政治団体は、内訳が必要（1万円超）<br>③上記以外の団体は、内訳・領収書は不要 |
| 2 政治活動費 | |
| （1）組織活動費 | 当該政治団体の組織に要する経費（選挙に関して支出されるものを除く。）で、例えば、当該団体の大会費、行事費、渉外費、外の政治活動を行う上で要する経費（選挙に関するものを除く。）、組織対策費（他団体との交渉、意見交換の経費）、交際費など（選挙に関するものを除く。） |
| （2）選挙関係費 | 選挙に関して支出される経費で、例えば、推薦料や陣中見舞など。その他選挙に関して行われる政治活動費等（選挙運動費等）の類似の支払いの経費、公認料・推薦料等を除く。）政治活動費に関する寄附（選挙に関するものを除く）については、「（5）寄附・交付金」に区分される。 |
| （3）ア 機関紙誌の発行事業費 | 刊行物、同関誌、荷造発送費、原稿料、発行事業緯行以外の政党等の発行に要する経費（選挙に関するものを除く。）で、例えば機関紙誌発行の内の政党等の公報広告に要する経費（選挙運動等）を除く。）の寄附 |
| （3）イ 宣伝事業費 | 遊説費、新聞・テレビ・ラジオ広告費、機関紙誌発行以外の政党等の公報広告に要する経費（選挙運動費等を除く。）で、例えばポスター・ビラ・パンフレット等の作成費、本ヘムペーシの作成費 |
| （3）ウ 政治資金パーティー開催事業費 | 政治資金パーティーの開催に要した経費をいうが、会場借上費、謝礼者への会費・売上と「3 機関紙誌の発行その他の事業による収入」に掲げる収入・事業に関し別離ごとに別葉でまとめる（継続した事業のために会費を徴収したもの。） |
| （3）エ その他の事業費 | 会費や売上と「3 機関紙誌の発行その他の事業による収入」に掲げる事業に要する経費、開催日ごとに別葉でまとめる（継続した事業のために会費を徴収したもの。） |
| （4）調査研究費 | 政治活動のために行う調査研究に要する経費で、例えば、研修会費、資料費、書籍購入費など |
| （5）寄附・交付金 | 寄附、交付金など |
| （6）その他の経費 | 借入金返済、寄付金など<br>政党及びその他の政治団体を問わず、「本部又は支部から供与された交付金に係る収入」又はその他の支出に対応する「金銭以外のもの」による寄附相当分）の額を問わず再掲する。金額を問わず、その名目は、金銭以外のものによる寄附相当 |

---

## 注記

（1）資金管理団体は人件費以外の一件当たりの金額が5万円以上の支出について、国会議員関係政治団体は人件費以外の一件当たり1万1円以上につき、その他の政治活動は政治活動の一件当たり月日等の明細及び支出の目的及び支出を受けた者の氏名及び住所（団体の場合は、名称及び主たる事務所の所在地）並びに支出月日等の支出について1組提出します。

（2）政治活動費は、上記の区分（例示）により、項目別区分小分類し、それぞれ明細又は「その他の政治資金団体」は監査意見書上領収書等の写し、「政党の本部又は国会議員関係政治団体」は政治資金監査報告書と領収書等の写しを提出する。なお、領収書等は不要（例えば、人件費や1組提出）

（3）「その他の政治資金団体」は監査意見書上領収書等の写しを提出し、収支報告書の写し（例えば、人件費や1組提出）

（4）支出項目の分類は、基本的に支出の目的により区分する（例えば、人件費でも宣伝事業として計上する。）。
