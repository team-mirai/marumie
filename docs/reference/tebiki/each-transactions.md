# 収支報告書様式の記載ルール（その3・その4）

出典: 政治資金報告書の手引き 11ページ（p.120）

## 1. 機関紙誌の発行その他の事業による収入（その3）

### 記載対象

会費等の収入を伴って行った事業等をすべて記載する。

### 記載項目

| 項目 | 内容 |
|---|---|
| 事業の種類 | 事業名を記載（例：機関紙「国政報告だより」、新宿太郎を励ます会、〇△記念パーティー、忘年会） |
| 金額 | 当該事業による収入金額 |
| 備考 | 開催年月日、開催場所等を記載 |

### 記載ルール

1. **（その3）に記載した事業は、（その2）の「2 政治活動費（3）機関紙誌の発行その他の事業費」の支出に対応していなければならない**
   - 必ず「アからエの各事業」に区分し、かつ事業の種類ごとに（その15）を作成する

2. **政治資金パーティーは、備考欄に開催日・開催場所等を記載しなければならない**

3. **特定パーティー（1,000万円以上の政治資金パーティー）の場合は、併せて（その10）にも記載しなければならない**

4. **都団体は、合計額が（その2）の「3 機関紙誌の発行その他の事業による収入」に記載の額と一致していなければならない**

5. **同一事業名で複数回開催した場合は、備考欄に開催日を記入しなければならない**

### 注記事項

- （注1）同一の事業収入は一行に計上する
- （注2）政治資金パーティーのうち、1,000万円以上のパーティーについては（その10）に詳録を作成する
- （注3）1回の政治資金パーティーにつき、計算から20万円超の対価の支払いがある場合は（その11）に支払をした者の氏名等を記載する
- （注4）政治資金パーティーは開催場所（開催施設の所在地（区市町村名を含む）及び施設名）を記載する

---

## 2. 借入金（その4）

### 記載対象

借入金がある場合に記載する。

### 記載項目

| 項目 | 内容 |
|---|---|
| 借入先 | 借入先の名称（例：新宿銀行新宿支店、新山 新一郎） |
| 金額 | 借入金額 |
| 備考 | 必要に応じて記載 |

### 記載ルール

1. **その年の12月31日現在で、借入先ごとの残高が100万円を超える場合（100万1円以上の場合）は、（その17）の「資産等の総括表」の「シ 借入先ごとの残高が100万円を超える借入金」の有の□にレ点を付けなければならない**

2. **上記に該当する場合は、（その18）に借入先及び未返済の金額を具体的に記載しなければならない**

3. **都団体は、合計額が（その2）の「4 借入金」に記載の額と一致していなければならない**

### 注記事項

- （注1）12月31日現在で、同一の借入先からの100万円を超える借入金がある場合は、（その17）に有と記載し、内訳として（その18）が必要
- （注2）合計は最終頁に記載する

---

## 3. 本部又は支部から供与された交付金に係る収入（その5）

※ここで言う支部とは、選挙管理委員会又は総務大臣へ届出をした支部を指す。

### 記載対象

政治団体の本部又は支部として届出のある団体が、当該政治団体の本部又はその他の支部から受けた交付金を記載する。

### 記載項目

| 項目 | 内容 |
|---|---|
| 交付金を供与した本部又は支部の名称 | 交付元の団体名（例：新宿政策研究会〇〇支部） |
| 金額 | 交付金額 |
| 年月日 | 交付を受けた年月日 |
| 主たる事務所の所在地 | 交付元団体の事務所所在地 |
| 備考 | 必要に応じて記載 |

### 記載ルール

1. **この様式は、政治団体の本部又は支部として届出のある団体が、当該政治団体の本部又はその他の支部から受けた交付金を記載するためのものである**

2. **本部又は支部を持たない独立した政治団体がこの様式を使用することはない**

3. **都団体は、合計額が（その2）の「5 本部又は支部から供与された交付金に係る収入」に記載の額と一致していなければならない**

### 注記事項

- （注1）同一の本部又は支部（選挙管理委員会又は総務大臣へ届け出たものに限る。）からの交付金が複数ある場合は「名寄せ」して年月日順に記載し、その者の最後に「計」を入れる
- （注2）合計は最終頁に記載する

---

## 4. その他の収入（その6）

### 記載対象

収入の各項目（その3〜5、その7）に区分されない収入がある場合に記載する。

### 記載項目

| 項目 | 内容 |
|---|---|
| 摘要 | 収入の内容（例：金銭以外のものによる寄附相当分、広告掲載料） |
| 金額 | 収入金額 |
| 備考 | 詳細情報（例：新宿太郎に選挙運動用葉書を提供、機関紙掲載広告料） |

### 記載ルール

1. **収入の各項目（その3〜5、その7）に区分されない収入がある場合にこの様式に記載しなければならない**

2. **支出（その14、その15）で無償提供したもの（例：選挙運動用葉書、選挙事務所等）がある場合には、必ずこの様式に収入として記載しなければならない**

3. **都団体は、（その2）の「6 その他の収入」に記載した内容と一致していなければならない**

4. **例えば「預金利息」等で、1件10万円に満たないものは合算して「1件10万円未満のもの」に記載しなければならない**

### 金額による記載区分

| 金額区分 | 記載方法 |
|---|---|
| 1件10万円以上 | 個別に記載 |
| 1件10万円未満 | 合算して「1件10万円未満のもの」欄に計金額を記載 |

### 注記事項

- （注1）1件10万円以上の収入は個別に記載し、1件10万円未満の収入は「1件10万円未満のもの」欄に一括してその合計金額を記載する
- （注2）「1件10万円未満のもの」と「合計」は最終頁に記載する

---

## 5. 寄附の内訳（その7）

出典: 政治資金報告書の手引き 13ページ（p.122）

### （1）個人からの寄附

#### 特定寄附について

「特」マークは、特定寄附であることの表示である。特定寄附とは、公職の候補者自身が政党から受けた寄附で、自ら指定した資金管理団体に寄附したものである。

#### 記載項目

| 項目 | 内容 |
|---|---|
| 寄附者の氏名（又は名称） | 寄附した個人の氏名 |
| 金額 | 寄附金額 |
| 年月日 | 寄附を受けた年月日 |
| 住所（又は所在地） | 寄附者の住所 |
| 職業（又は代表者の氏名） | 寄附者の職業 |
| 備考 | 必要に応じて記載（例：遺贈、事務所の無償提供） |

#### 記載ルール

1. **年間5万円を超える（5万1円以上）寄附者について、個別に記載しなければならない**

2. **記載する内容は、寄附した時点（年月日）における氏名、住所及び職業でなければならない**

3. **寄附金控除を受けるときには、年間5万円以下であっても記載しなければならない**

4. **遺贈によってする寄附については、備考欄に「遺贈」と記載しなければならない**

5. **同一の寄附者から複数回の寄附がある場合は、小計を記載しなければならない**

#### 複数ページにまたがる場合

- 複数ページの「小計」と「その他の寄附」の合計額を記載する
- 上記内訳に記載した以外の寄附額の合計は、記載が複数ページになる場合は最後のページに記載する

#### 注記事項

- （注1）同一者からの年間5万円超（5万1円以上）の寄附は個別に記載する
- （注2）同一者から複数回寄附を受けた場合は、寄附をした者（団体等）ごとに「名寄せ」して年月日順に記載し、その者の最後に「計」を入れる
- （注3）「その他の寄附」と「合計」の欄は、個人、法人・その他の団体又は政治団体の寄附者の区分ごとに、最終頁に記載する
- （注4）当該政治団体の本部又は支部（選挙管理委員会又は総務大臣へ届け出たものに限る。）からの交付金は、（その5）に記載する

---

### （2）法人その他の団体からの寄附

※政党・政党の支部及び政治資金団体以外にはありません。

#### 寄附者の区分

記載時に以下のいずれかに○をつける：
1. 個人
2. 法人・その他の団体
3. 政治団体

#### 記載項目

| 項目 | 内容 |
|---|---|
| 寄附者の氏名（又は名称） | 法人・団体の名称（例：D株式会社、株式会社E） |
| 金額 | 寄附金額 |
| 年月日 | 寄附を受けた年月日 |
| 住所（又は所在地） | 法人・団体の所在地 |
| 職業（又は代表者の氏名） | 代表者の氏名 |
| 備考 | 必要に応じて記載（例：上場・外資50%超） |

#### 記載ルール

1. **年間5万円を超える（5万1円以上）寄附について、個別に記載しなければならない**

2. **外国法人又はその主たる構成員が外国人若しくは外国法人である団体その他の組織から政治活動に関する寄附を受けることはできない**
   - ただし、その主たる構成員が外国人又は外国法人のうち上場会社であって、その発行する株式が証券取引所において5年以上継続して上場されているものからの寄附は受けることができる
   - その場合は備考欄に「上場・外資50%超」と記載する

3. **法人その他の団体から寄附を受けることができるのは、政党・政党の支部及び政治資金団体に限られる**

---

## 6. 寄附のうち寄附のあっせんによるものの内訳（その8）

出典: 政治資金報告書の手引き 14ページ（p.123）

### 記載項目

| 項目 | 内容 |
|---|---|
| 寄附のあっせん者の氏名（又は名称） | あっせんした者の氏名・名称（例：K政治連盟） |
| 金額 | あっせんによる寄附金額 |
| 提供年月日 | 寄附を受けた年月日 |
| 集めた期間 | あっせんにより寄附を集めた期間 |
| 住所（又は所在地） | あっせん者の住所・所在地 |
| 職業（又は代表者の氏名） | あっせん者の職業・代表者氏名 |
| 備考 | 必要に応じて記載 |

### あっせん者の区分

記載時に以下のいずれかに○をつける：
1. 個人
2. 法人・その他の団体
3. 政治団体

### 記載ルール

1. **都団体は、（その2）の「2 寄附（寄附のうち寄附のあっせんによるもの）」に記載の額と一致していなければならない**

### 注記事項

- （注）あっせんをした者（団体等）ごとに「名寄せ」して年月日順に記載し、その者の最後に「計」を入れる

## 7. 政党匿名寄附の内訳（その9）

※政党・政党の支部及び政治資金団体以外にはありません。

### 記載対象

政党匿名寄附は、政党、政党の支部及び政治資金団体が、街頭又は一般に公開される演説会若しくは集会の会場において受けた1件1,000円以下の匿名寄附を指す。

### 記載項目

| 項目 | 内容 |
|---|---|
| 政党匿名寄附を受けた場所 | 受領場所（例：〇〇区〇〇町〇丁目 〇〇駅前街頭） |
| 金額 | 寄附金額 |
| 年月日 | 寄附を受けた年月日 |
| 備考 | 必要に応じて記載（例：〇〇演説会） |

### 記載ルール

1. **政党匿名寄附は、政党、政党の支部及び政治資金団体が、街頭又は一般に公開される演説会若しくは集会の会場において受けた1件1,000円以下の匿名寄附でなければならない**

2. **この寄附は、政党、政党の支部及び政治資金団体以外は受領できないので、その他の政治団体にはこの様式はない**

### 注記事項

- （注）政党、政党の支部及び政治資金団体が、街頭又は一般に公開される演説会の会場等で受けた1,000円以下の匿名寄附が対象となる

## 8. 政治団体からの寄附（その7の続き）

出典: 政治資金報告書の手引き 14ページ（p.123）

### 記載ルール

1. **当該政治団体と本部支部の関係にない政治団体（政党）の支部等は、交付金に係る収入（その5）ではなく、寄附（その7）に記載しなければならない**

### 注記事項

- （注1）同一者からの年間5万円超（5万1円以上）の寄附は個別に記載する
- （注2）同一者から複数回寄附を受けた場合は、寄附をした者（団体等）ごとに「名寄せ」して年月日順に記載し、その者の最後に「計」を入れる
- （注3）「その他の寄附」と「合計」の欄は、個人、法人・その他の団体又は政治団体の寄附者の区分ごとに、最終頁に記載する
- （注4）当該政治団体の本部又は支部（選挙管理委員会又は総務大臣へ届け出たものに限る。）からの交付金は、（その5）に記載する

## 9. 特定パーティーの対価に係る収入の内訳（その10）

出典: 政治資金報告書の手引き 15ページ（p.124）

### 記載対象

（その3）に記載した政治資金パーティーの中から、特定パーティー（収入が1,000万円以上の政治資金パーティー）又は特定パーティーとなると見込まれるものについて、再度詳細を記載する。

### 記載項目

| 項目 | 内容 |
|---|---|
| 特定パーティーの名称 | パーティー名（例：新宿太郎を励ます会、〇△記念パーティー） |
| 対価に係る収入の金額 | 収入金額 |
| 対価の支払をした者の数 | 支払者数 |
| 開催年月日 | 開催日 |
| 開催場所 | 開催施設の所在地・施設名 |
| 備考 | 前年収入、共催団体の名称・金額等 |

### 記載ルール

1. **（その3）に記載した政治資金パーティーの中から、特定パーティー（収入が1,000万円以上の政治資金パーティー）又は特定パーティーとなると見込まれる政治資金パーティーは、再度詳細を記載しなければならない**

2. **前年開催又は翌年開催であっても、開催規模が1,000万円以上又は1,000万円以上と見込まれる政治資金パーティーは、当該年の収入が1,000万円未満であっても記載しなければならない**

3. **前年の収入がある場合には、備考欄にその人数・金額を記載しなければならない**

4. **共催の場合には、備考欄に共催した団体の名称・金額を記載しなければならない**

5. **都団体は、合計額が（その2）の「3 機関紙誌の発行その他の事業による収入（1）（3のうち特定パーティーの対価に係る収入）」に記載の額と一致していなければならない**

### 注記事項

- （注1）特定パーティーとは対価に係る収入の金額が1,000万円以上の政治資金パーティーをいう
- （注2）開催場所欄には、開催施設の所在地（区市町村名を含む）及び施設名を記載する
- （注3）（その3）に記載した特定パーティーの詳細を再掲する

---

## 10. 政治資金パーティーの対価に係る収入の内訳（その11）

### 記載対象

（その3）の事業で政治資金パーティーを開催した団体のうち、1回のパーティーにつき「1人又は1法人等」で「20万円を超える」パーティー券の購入者があった場合に記載する。

### 記載項目

| 項目 | 内容 |
|---|---|
| 政治資金パーティーの名称 | パーティー名（例：新宿太郎を励ます会） |
| 対価の支払をした者の氏名（又は名称） | 支払者の氏名・名称 |
| 金額 | 支払金額 |
| 年月日 | 支払年月日 |
| 住所（又は所在地） | 支払者の住所・所在地 |
| 職業（又は代表者の氏名） | 支払者の職業・代表者氏名 |
| 備考 | 前年分の購入金額等 |

### 対価の支払をした者の区分

記載時に以下のいずれかに○をつける：
1. 個人
2. 法人・その他の団体
3. 政治団体

### 記載ルール

1. **同一の政治資金パーティーにおいて、20万円を超える（20万1円以上の）パーティー券の購入者について、パーティー別、購入者の区分別に記載しなければならない**

2. **前年にも当該政治資金パーティー券を購入しており、本年の購入分とあわせて20万円を超えた場合には、備考欄に「前年分の購入金額」を記載しなければならない**

### 注記事項

- （注）この頁には、1パーティーにつき、同一者からの対価の支払いが、20万円超（20万1円以上）のもののみ個別に記載する。合計は最終頁に記載する

## 12. 支出項目別金額の総括表（その13）

出典: 政治資金報告書の手引き 16ページ（p.125）

※都団体の支出項目別金額は、（その2）を参照すること。

1. **（その7）の金銭以外の寄附（事務所の無償提供など）に対応する支出の記載例である**

2. **金銭の収受がなく、通常領収書が発行されないので、「領収書等を徴し難かった支出の明細書」（第15号様式）に記載して添付しなければならない**

## 15. 本部又は支部に対して供与した交付金に係る支出の内訳（その16）

出典: 政治資金報告書の手引き 21ページ（p.130）

### 記載対象

（その14、その15）のうち、該当するものを再掲する。

### 記載項目

| 項目 | 内容 |
|---|---|
| 支出項目 | 支出の項目別区分（例：組織活動費、寄附・交付金） |
| 該当する支出の「項目別区分」 | 該当する支出を記載する |
| 金額 | 支出金額 |
| 月日 | 支出月日 |
| 交付金の供与を受けた本部又は支部の名称 | 本部又は支部の名称（例：新宿政策研究会〇〇支部、△△支部、〇〇支部） |
| 主たる事務所の所在地 | 本部又は支部の事務所所在地 |
| 備考 | 必要に応じて記載 |

### 記載ルール

1. **すべての支出のうち、政治団体の本部が支部（届出のあるものに限る。）へ、又は支部が本部やその他の支部（届出があるものに限る。）へ支出した場合に、その内容を記載しなければならない**

2. **この様式は、1円以上のすべての支出を記載しなければならない**

3. **全国団体は、支出項目にしたがって、（その13）の備考欄にその合計を記載しなければならない**

### 注記事項

- （注1）当該政治団体の本部又は支部（選挙管理委員会又は総務大臣へ届け出たものに限る。）への支出があった場合、そのすべて、（その14、その15）に記載したうえで、支出項目を問わず1円以上のすべての支出を再掲するものである。合計は最終頁に記載する
- （注2）支出項目は、その2（都団体）、または、その13（全国団体）の項目と一致する
  - 例：組織活動費、寄附・交付金
