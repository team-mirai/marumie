# 政治資金報告書XML形式差分分析

## 概要

`minimum_valid.xml`（公式マクロで作成した最小構成XML）と`report_3_2025.xml`（現在のseedデータから生成したXML）を比較し、フォーマットの差異を分析した。

## 比較対象ファイル

| ファイル | 説明 |
|----------|------|
| `data/reports/minimum_valid.xml` | 政治資金報告書作成マクロで作成した最小構成XML（正） |
| `data/reports/report_3_2025.xml` | 現在のseedデータから生成したXML（誤） |

## 発見された問題点

### 1. SYUUSHI07_01のSHEETタグ欠落

SYUUSHI07_01（第14号様式 その1：団体基本情報）の内容が`<SHEET>`タグで囲まれていない。

**正しい形式（minimum_valid.xml）:**
```xml
<SYUUSHI07_01>
  <SHEET>
    <HOUKOKU_NEN>2025</HOUKOKU_NEN>
    <KAISAI_DT></KAISAI_DT>
    <DANTAI_NM>サンプル党</DANTAI_NM>
    ...
  </SHEET>
</SYUUSHI07_01>
```

**現在の形式（report_3_2025.xml）:**
```xml
<SYUUSHI07_01>
  <HOUKOKU_NEN>2025</HOUKOKU_NEN>
  <KAISAI_DT/>
  <DANTAI_NM>サンプル党</DANTAI_NM>
  ...
</SYUUSHI07_01>
```

### 2. SYUUSHI_UMU_FLGの構造が異なる

様式有無フラグの親タグ構造が異なる。

**正しい形式（minimum_valid.xml）:**
```xml
<SYUUSHI_UMU_FLG>
  <SYUUSHI_UMU>110000000000000000001000000000000010000000000000100</SYUUSHI_UMU>
</SYUUSHI_UMU_FLG>
```

**現在の形式（report_3_2025.xml）:**
```xml
<SYUUSHI_FLG>
  <SYUUSHI_UMU_FLG>
    <SYUUSHI_UMU>101111000000011000000000000000000000000000000000000</SYUUSHI_UMU>
  </SYUUSHI_UMU_FLG>
</SYUUSHI_FLG>
```

余分な`<SYUUSHI_FLG>`タグで囲まれている。

### 3. 必須様式の欠落

以下の様式が`report_3_2025.xml`に存在しない：

| 様式 | 説明 | 影響 |
|------|------|------|
| **SYUUSHI07_02** | 収支総括表 | **致命的** - 収入総額・支出総額・繰越額が表示されない |
| SYUUSHI07_07 | 寄附の明細（個人・法人・政治団体） | 寄附内訳が表示されない |
| SYUUSHI07_08 | 寄附のあっせん明細 | - |
| SYUUSHI07_09 | 政党匿名寄附明細 | - |
| SYUUSHI07_10 | 政治資金パーティー収入明細 | - |
| SYUUSHI07_11 | パーティー対価支払者明細 | - |
| SYUUSHI07_12 | パーティー対価あっせん者明細 | - |
| SYUUSHI07_13 | 支出の総括表 | 支出の項目別集計が表示されない |
| SYUUSHI07_16 | 本部・支部への交付金支出明細 | - |
| SYUUSHI07_17 | 資産等の項目別内訳の有無 | 資産状況が表示されない |
| SYUUSHI07_18 | 資産等の項目別内訳明細 | - |
| SYUUSHI07_19 | 不動産使用状況明細 | - |
| SYUUSHI07_20 | 宣誓書 | - |
| SYUUSHI08 | 訂正等届出書（第15号様式） | - |
| SYUUSHI08_02 | 解散届出書（第16号様式） | - |
| SYUUSHI_KIFUKOUJYO | 寄附金控除団体情報 | - |

### 4. VERSIONの差異

| ファイル | VERSION |
|----------|---------|
| minimum_valid.xml | 20251001 |
| report_3_2025.xml | 20081001 |

互換性には影響しない可能性あり。

## 影響

- **基本情報が認識されない**: SYUUSHI07_01の`<SHEET>`タグ欠落により、団体名・代表者情報などが正しくパースされない可能性がある
- **様式有無フラグが認識されない**: `<SYUUSHI_FLG>`の余分なネストにより、どの様式にデータがあるかが判定できない
- **集計サマリーが表示されない**: SYUUSHI07_02（収支総括表）がないため、収入・支出の合計金額が表示されない
